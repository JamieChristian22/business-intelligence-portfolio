# Business Intelligence Analyst Portfolio (Dashboard-Matched)

This portfolio contains **5 real-world BI case studies aligned to your provided dashboards** (Power BI + Tableau).  
Each project includes:
- The **original dashboard file** (PBIX/TWBX)
- Reproducible **datasets (CSV)**
- Warehouse-ready **SQL schema + analysis queries**
- **DAX / Tableau calculated fields** documentation
- **Python KPI validation** scripts

## Projects
1. `01_Sales_Performance_Dashboard_PowerBI`
2. `02_Ecommerce_Product_Analytics_PowerBI`
3. `03_Amazon_Marketplace_Analytics_PowerBI`
4. `04_FPNA_Executive_Analytics_PowerBI`
5. `05_StreamFlow_SaaS_Product_Analytics_Tableau`

## How to use (fast)
- Open the dashboard file (Power BI Desktop / Tableau)
- In each project, load the CSVs from `data/`
- Review `docs/` for measures/calculated fields
- Run `python/kpi_validation.py` to verify KPIs
